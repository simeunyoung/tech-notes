# Dev Blog

실무에서 겪은 문제 해결 과정과 시행착오를 기록하기 위한 개인 개발 블로그입니다.

## Purpose
- 강의형 정리 글이 아닌, 실제 문제 상황과 해결 과정을 남기기 위함
- 서버/백엔드/운영 환경에서 발생한 이슈를 재현 가능한 형태로 기록
- 개인 학습 기록을 장기적으로 자산화

## Content
- Spring Boot / Backend
- Server / Infra / DevOps
- Logging / Troubleshooting
- 설정 이슈, 장애 원인 분석, 개선 과정

## Stack
- Next.js
- Markdown
- Deployed on Vercel

## Notes
- 모든 글은 개인 프로젝트 또는 실무 경험을 기반으로 작성
